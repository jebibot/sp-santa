구독 랜덤 선물 SOOP에서 구독권을 랜덤으로 선물합니다.

1. 메뉴 > 북마크 > '북마크바 표시'를 활성화합니다.

2. 아래 버튼을 북마크바로 드래그하여 북마크로 추가합니다.

구독권 랜덤 선물 ← 이 버튼을 북마크바로 드래그 하세요.

3. 티어1 1개월 구독 선물권을 선물한 인원수만큼 구매합니다.

4. 구독권을 선물할 방송에 들어갑니다.

5. 북마크를 클릭한 후 선물할 대상을 콤마로 구분하여 입력합니다.

6. 선물할 인원수를 입력합니다.

본 서비스는 SOOP과 관련이 없으며 관련 상표는 각 소유자의 등록된 상표입니다. 다량의 구독권을 일시에 선물하면 오류가 발생할 수 있으며, 본 서비스를 사용하여 발생하는 결과에 대한 모든 책임은 사용자에게 있습니다.
